## 여행장소

**삿포로**(일본어: 札幌市)는 일본 홋카이도의 도청 소재지이자 정치, 경제의 중심 도시다. 삿포로에 대한 자세한 내용은 [여기](https://ko.wikipedia.org/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C_%EC%8B%9C)서 볼 수 있다.

	* 숙박 : 3월 1일 ~ 2일(1박)

	* 체크아웃 : 11:00
